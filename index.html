Features

Auctions + Orders
Sell + Shop
Homes + Warps

Auction + Order:

Mit dem Auction & Order Plugin können Spieler Items versteigern, direkt verkaufen oder spezielle Bestellungen aufgeben.

Du willst dein seltenes Schwert teuer verkaufen? Oder ein bestimmtes Item von anderen Spielern bekommen? Kein Problem!

Das Plugin macht den Handel einfach, übersichtlich und fair. So wird der Servermarkt lebendig, Preise passen sich an und jeder kann am Handel teilnehmen.

Sell + Shop:

Mit dem Sell & Shop System können Spieler Items direkt verkaufen oder ausgewählte Items im Shop kaufen.

Du hast überschüssige Materialien oder seltene Items? Verkaufe sie über Sell und erhalte sofort Ingame-Geld! Oder suchst du nach spezieller Ausrüstung und besonderen Ressourcen? Im Shop findest du ausgewählte Items, die du direkt erwerben kannst.

Das System ist einfach, übersichtlich und fair gestaltet. So können Spieler jederzeit handeln, ihr Inventar sinnvoll nutzen und der Servermarkt bleibt lebendig und stabil.

Home + Warp:

Mit dem Home & Warp System können Spieler sich jederzeit zu ihrem Zuhause oder zu speziellen Orten auf dem Server teleportieren.

Du willst schnell zu deinem Spawn, zu deinen Crates oder anderen wichtigen Punkten gelangen? Setze Homes oder nutze Warps und spare dir lange Wege!

Das System ist einfach zu bedienen, zuverlässig und praktisch. So kannst du dich frei bewegen, deine Lieblingsorte schnell erreichen und das Spielerlebnis noch angenehmer gestalten.

Wie man beitritt

Text Turorial

Starte Minecraft Java Edition.

Klicke im Hauptmenü auf Multiplayer.

Beim ersten Öffnen erscheint eine Warnmeldung – bestätige sie mit „Ich verstehe“.

Klicke auf Server hinzufügen.

Trage bei Server-Name einen beliebigen Namen ein.

Trage bei Server-Adresse die IP: mc.schnur.net ein.

Speichern → Server anklicken → Beitreten.

Clicke den NPC an oder verwende den Befehl /worlds um die Welt zu betreten.
